TFT_eSPI Simulator es una herramienta de escritorio gratuita para programadores de Arduino y ESP32 que permite visualizar en tiempo real cómo lucirá su código en una pantalla TFT física, sin necesidad de cargar el programa al microcontrolador ni tener el hardware conectado.

Diseñado específicamente para trabajar con la popular librería TFT_eSPI de Bodmer, el simulador interpreta directamente los comandos de dibujo y texto del código fuente, mostrando una previsualización interactiva y precisa del display.

¿Para quién es útil? Para cualquier persona que programe pantallas TFT con Arduino, ESP8266 o ESP32 y quiera diseñar interfaces gráficas de forma rápida y eficiente, ahorrando tiempo de compilación y cargas repetidas al hardware.

Funciones principales:

— Editor de código integrado con resaltado de sintaxis compatible con Arduino IDE, incluyendo coloreado de comandos TFT, cadenas de texto, números, comentarios y palabras clave.

— Previsualización en tiempo real que se actualiza automáticamente mientras se escribe el código, sin necesidad de presionar ningún botón.

— Soporte completo de la API TFT_eSPI, incluyendo fillScreen, drawPixel, drawLine, drawFastVLine, drawFastHLine, drawRect, fillRect, drawRoundRect, fillRoundRect, drawCircle, fillCircle, drawEllipse, fillEllipse, drawTriangle, fillTriangle, drawArc, drawSmoothArc, drawBitmap, pushImage, print, println, drawString, drawCentreString, drawRightString, drawChar, drawNumber y drawFloat.

— Simulación de rotación de pantalla en los 4 modos estándar (0°, 90°, 180° y 270°), con intercambio automático de dimensiones.

— Resolución de pantalla configurable para simular cualquier modelo de display TFT, desde los 128×128 hasta los 480×320 y superiores.

— Zoom interactivo de 25% hasta 500% para inspeccionar detalles del diseño con precisión.

— Paleta completa de colores TFT_eSPI con más de 30 constantes de color predefinidas, soporte de valores RGB565 en hexadecimal y la función color565(r, g, b).

— Soporte de símbolos especiales mediante notación (char)N, incluyendo el símbolo de grado °, libra esterlina £ y otros caracteres ASCII.

— Panel de log en tiempo real que muestra errores y advertencias de cada línea del código con número de línea exacto, facilitando la depuración.

— Indicador de coordenadas que muestra la posición exacta del cursor en píxeles de pantalla al mover el mouse sobre la previsualización.

— Apertura y guardado de archivos .ino, .cpp y .h directamente desde el editor.

— Ventana de referencia rápida con la lista completa de comandos soportados, formatos de color y símbolos especiales, accesible desde la barra de herramientas.
